Tags

  1. Hyperlinks Tag- HTML links are hyperlinks. You can click on a link and jump to another document.
    • Absolute URLs
    • Relative URLs
    • Syntax:< a href="url">link text < /a>
  2. Image Tag- Images can improve the design and the appearance of a web page.
    • The HTML < img /> tag is used to embed an image in a web page.
    • The < img > tag has two required attributes:
      1. src- Specifies the path to the image
      2. alt- Specifies an alternate text for the image
      3. Syntax:< img src="pic_trulli.jpg" alt="Italian Trulli">
  3. Video- The HTML < video> element is used to show a video on a web page.
  4. Iframe- An HTML iframe is used to display a web page within a web page.
